Teams Icon Generator

Generate Microsoft Teams app package icons following official specifications: 192x192 color icon with 48px padding and 32x32 white outline icon with transparent background.

Color Icon
192x192px • 96x96 symbol • 48px padding
Outline Icon
32x32px • RGB(255,255,255) • transparent
Teams Specifications: Color icon must be square with solid/transparent background. Outline icon must use RGB(255,255,255) white with transparent background. Teams automatically rounds corners.

How to Save Icons

  1. Click "Generate Icons" above to create the icons
  2. Right-click on the Color Icon canvas → "Save image as..." → Save as icon-color.png
  3. Right-click on the Outline Icon canvas → "Save image as..." → Save as icon-outline.png
  4. Save both files in the package-for-teams directory
  5. Run the package creation script: cd package-for-teams && zip inclusively-teams-app.zip manifest.json icon-color.png icon-outline.png
